# AI 算命 H5 应用设计方案 - "灵机 (CyberDivination)"

## 1. 产品概述

一款结合传统东方玄学与现代 AI 技术的 H5 应用。通过沉浸式的交互体验和独特的“新中式赛博”视觉风格，为用户提供娱乐性的算命体验。

## 2. 视觉风格 (Visual Identity)

**主题**: 新中式赛博 (Neo-Chinese Cyber)

- **核心理念**: 当古老的易经算法遇上神经网络。
- **配色**:
  - 背景: 深邃墨黑 (`#050505`) - 象征虚空与未知。
  - 主色: 朱砂红 (`#FF2A2A`) - 象征能量与运势。
  - 辅色: 荧光青 (`#00FFCC`) - 象征数据与灵光。
  - 纹理: 低透明度的电路板纹理叠加传统云纹。
- **字体**: 标题使用具有书法韵味的字体（如 Ma Shan Zheng 或类似风格 Web Font），正文使用现代无衬线体 (Inter/Roboto)。
- **动效**:
  - 呼吸灯效果。
  - 粒子流动的八卦阵。
  - 类似“黑客帝国”代码雨，但由干支纪年字符组成。

## 3. 功能模块

### 3.1 启动页 (Splash)

- **画面**: 一个旋转的赛博太极图，周围环绕着流动的二进制代码和八卦符号。
- **交互**: 点击“开启灵机”进入。

### 3.2 信息录入 (Input)

- **表单**:
  - 姓名 (Name)
  - 生辰 (Date of Birth & Time) - 使用滚轮选择器，模拟罗盘转动。
  - 性别 (Gender)
  - 问询方向 (Query): 综合运势、事业、爱情、财运。
- **交互**: 输入时伴随机械键盘音效（可选），确认后触发“注入数据”动画。

### 3.3 计算过程 (Processing)

- **画面**:
  - 用户的生辰八字被转化为十六进制代码。
  - AI 神经网络节点连接，形成星盘形状。
  - 屏幕中央出现“正在通灵...”或“Connecting to Destiny Matrix...”。
- **时长**: 约 3-5 秒，建立期待感。

### 3.4 结果展示 (Result)

- **卡片式布局**: 适合截图分享。
- **内容**:
  - **灵签/判词**: 四句诗或者一段玄妙的话。
  - **核心指数**: 雷达图展示各项运势（五行能量分布）。
  - **AI 详细解读**: 基于 LLM 生成的个性化运势分析。
  - **今日宜忌**: 结合现代场景（例如：宜 Code Review，忌 部署上线）。
- **操作**: “保存灵符”（生成图片）、“分享天机”。

## 4. 技术栈

- **框架**: React (Vite)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **动画**: Framer Motion (关键，用于实现复杂的仪式感动画)
- **路由**: React Router
- **图标**: Lucide React / React Icons
- **构建工具**: Vite

## 5. 开发计划

1. **初始化**: 搭建 Vite + React + TS 项目环境，配置 Tailwind。
2. **基础组件**: 开发通用 UI 组件 (Button, Input, Card) 并应用赛博风格。
3. **页面实现**: 依次开发 Splash, Input, Processing, Result 页面。
4. **逻辑对接**: 模拟 AI 接口返回算命结果。
5. **优化**: 调整动画流畅度，适配移动端。
